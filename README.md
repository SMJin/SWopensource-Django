# Installing Django

```
py -m pip install Django
```

# make Django Project

```
django-admin startproject mydjangoproject
```

# make Django App

```
py manage.py startapp mydjangoapp
```

# start Django Server

```
py manage.py runserver
```

# migrate Django Models

```
py manage.py makemigraion myappmodels
py manage.py migrate
```
